## Features
- 🎮 Classic Pong gameplay.
- 👫 Two-player mode.
- 🏆 Score tracking.
- 🔄 Game restart feature.

## Controls
- **Player 1 (Blue Paddle)**: `W` (Up), `S` (Down)
- **Player 2 (Red Paddle)**: `Up Arrow` (Up), `Down Arrow` (Down)
- 🔁 **Restart Game**: Press `R`
